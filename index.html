This is a simple experiment using Three.js and WebGL to animate 3d models. The way this works is that there is an endless loop operating as a "tick".
Each "tick" it requests that we move to the next animation frame. The models can be thought of as gifs. A collection of images, each of which is a frame, and there is some time mechanism used to go from frame to frame.
